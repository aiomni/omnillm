# Product Requirements Document: omnillm Website ## Project Overview Design a high-performance, developer-centric website for `omnillm`, a Rust library for LLM interactions. The aesthetic should be minimalist, modern, and "geeky," reflecting the technical excellence of the Rust ecosystem. ## Target Audience - Rust developers - AI/ML engineers - Open-source contributors ## Key Features & Pages 1. **Landing Page (Hero)**: High-impact introduction, clear value proposition, quick install command, and a "Hello World" code snippet. 2. **Features/Architecture**: Explaining why `omnillm` is different (performance, type safety, multi-model support). 3. **Getting Started/Docs Preview**: A clean, readable documentation layout with syntax highlighting. 4. **Examples/Showcase**: Practical code examples demonstrating the library's capabilities. ## Visual Direction (Omni Tech Dark) - **Theme**: Dark Mode - **Typography**: Space Grotesk (Headings) & Monospace (Code/Body) - **Accents**: High-contrast tech blue or orange. - **Layout**: Clean grid-based structure with minimal distractions.
